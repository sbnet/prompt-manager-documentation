Prompt Manager propose deux plans adaptés à vos besoins.

## Offre FREE (Gratuite)

Idéal pour découvrir la plateforme et pour les projets personnels.

**Inclus :**
- Gestion illimitée de prompts (création, édition).
- Versioning complet.
- Accès aux modèles gratuits.

**Limites :**
- **3 Exécutions / jour** (Via UI ou API).
- Pas de fonctionnalité BYOK.

## Offre PRO (7$ / mois)

Pour les professionnels et les équipes qui ont besoin de puissance et de flexibilité.

**Avantages :**
- **Exécutions Illimitées** : Plus de quota journalier sur l'interface.
- **Accès Prioritaire** : Support client prioritaire.
- **Modèles Premium** : Accès aux modèles les plus performants (GPT-4, Opus, etc.).
- **BYOK (Bring Your Own Key)** : Connectez votre propre compte OpenRouter.

### Le BYOK (Bring Your Own Key)

Le BYOK permet aux utilisateurs PRO d'utiliser leur propre clé API OpenRouter.

**Pourquoi ?**
- **Confidentialité** : Vous maîtrisez vos données.
- **Coûts Réels** : Vous payez directement le fournisseur (OpenRouter) au prix coûtant, sans surcoût.
- **Pas de Limites** : Vous n'êtes limité que par votre budget OpenRouter.

**Configuration :**
1. Allez dans **Settings > BYOK**.
2. Entrez votre clé OpenRouter (`sk-or-...`).
3. Sauvegardez.
