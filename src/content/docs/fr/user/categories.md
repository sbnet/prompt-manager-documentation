Les catégories vous permettent d'organiser vos prompts de manière hiérarchique pour une meilleure gestion de vos projets.

## Créer une Catégorie

1. Dans la section **Prompts**, cliquez sur le bouton **Nouvelle Catégorie**.
2. Donnez un nom à votre catégorie.
3. (Optionnel) Sélectionnez une catégorie parente pour créer une sous-catégorie.

## Catégories Hiérarchiques

Vous pouvez créer des structures de catégories imbriquées :

```
Marketing
  ├─ Emails
  ├─ Réseaux Sociaux
  └─ Blog
Support Client
  ├─ Réponses Automatiques
  └─ Escalade
```

Cette organisation facilite la navigation dans des bibliothèques de prompts volumineuses.

## Assigner un Prompt à une Catégorie

Lors de la création ou de l'édition d'un prompt, vous pouvez :
- Sélectionner une catégorie existante dans le menu déroulant
- Créer une nouvelle catégorie à la volée
- Laisser le prompt sans catégorie

## Filtrer par Catégorie

Dans la liste des prompts, utilisez le filtre de catégorie pour afficher uniquement les prompts d'une catégorie spécifique.

:::tip[Bonne Pratique]
Organisez vos prompts par domaine fonctionnel (Marketing, Support, Produit) ou par projet pour faciliter la collaboration en équipe.
:::
