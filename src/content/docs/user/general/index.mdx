Bienvenue sur la documentation utilisateur de **Prompt Manager**.

## Qu'est-ce que Prompt Manager ?

**Prompt Manager** est une plateforme complète conçue pour industrialiser l'usage des modèles de langage (LLM) au sein de vos applications. Elle s'adresse aux développeurs, aux prompt engineers et aux équipes produit.

### Fonctionnalités Clés

- **Versioning Immuable** : Gérez le cycle de vie de vos prompts (DRAFT → PRODUCTION → ARCHIVED) en toute sécurité. Chaque modification crée une nouvelle version, garantissant qu'aucune régression n'affecte vos applications en production.
- **Support Multi-Modèles** : Accédez à une large gamme de modèles via OpenRouter (OpenAI, Anthropic, Mistral, Llama, etc.) sans changer votre code.
- **Prompt IDE Avancé** : Éditeur professionnel à trois panneaux avec configuration complète du modèle et test en temps réel avec streaming.
- **Organisation par Catégories** : Structurez vos prompts en catégories hiérarchiques pour une gestion optimale.
- **Analytics & Suivi** : Surveillez l'utilisation, les performances et les coûts de tous vos prompts en temps réel.
- **API Unifiée (PRO)** : Intégrez vos prompts dans vos applications via une API REST simple et performante.
- **BYOK (Bring Your Own Key)** : Pour les utilisateurs PRO, utilisez votre propre clé API OpenRouter pour un contrôle total de la facturation et de la confidentialité.

## Pour qui ?

- **Développeurs** : Intégrez l'IA générative sans gérer la complexité des différents fournisseurs de modèles.
- **Product Owners** : Itérez sur les prompts et améliorez la qualité des réponses sans toucher au code.
- **QA / Testeurs** : Validez les performances des prompts avant la mise en production.
