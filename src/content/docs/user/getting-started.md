Ce guide vous aidera à démarrer rapidement avec Prompt Manager.

## Création de Compte

Pour commencer à utiliser Prompt Manager, vous devez disposer d'un compte.
L'inscription est simple et rapide. Une fois inscrit, vous accéderez automatiquement au plan **FREE**.

## Interface Utilisateur

L'interface est découpée en plusieurs sections principales :

### Dashboard
Votre tableau de bord principal affichant votre consommation récente et l'état de vos quotas.

### Prompts
L'espace central où vous créez, éditez et gérez vos prompts. C'est ici que vous passerez le plus clair de votre temps.

### Settings (Paramètres)
Gérez vos préférences, votre abonnement et votre configuration BYOK (pour les utilisateurs PRO).

## Votre Premier Prompt

1. Naviguez vers la section **Prompts**.
2. Cliquez sur **New Prompt**.
3. Donnez un nom et une description à votre prompt.
4. Vous serez redirigé vers l'éditeur (Prompt IDE).
